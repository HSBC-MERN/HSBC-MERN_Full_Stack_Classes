components In React JS ðŸ‘
--------------------------
    ->Smallest UI Block in Web Page 

    Types components : 
        1. Function based component (16.8)

             Syntax: 
                function Nav(){
                        return (
                            <div>
                                 <h1>THIS NAV Component</h1>
                            </div>
                                )
                            }
                export default Nav;


            ->FBC is stateless
            ->super and this keyword is not present
            ->render() function not present


        2. Class Based component 

        Syntax : 
        import { Component } from "react";
        class Nav extends Component{
            render(){
                return(
                    <h1>CBC</h1>
                )
            }
        }
        export default Nav;

        
            ->CBC is stateFull Component 
            ->super and this keyword is  present
            ->render() function  present
