    Types components : 
        1. Function based component (16.8)

             Syntax: 
                function Nav(){
                        return (
                            <div>
                                 <h1>THIS NAV Component</h1>
                            </div>
                                )
                            }
                export default Nav;


            ->FBC is stateless
            ->super and this keyword is not present
            ->render() function not present


        2. Class Based component 

        Syntax : 
        import { Component } from "react";
        class Nav extends Component{
            render(){
                return(
                    <h1>CBC</h1>
                )
            }
        }
        export default Nav;

        
            ->CBC is stateFull Component 
            ->super and this keyword is  present
            ->render() function  present

MongoDB : 
----------
    ->MongoDB is the document oriented database used to store and manage the data 
    ->if semiStructure , unstructured 
    ->MongoDB flexible schema/ schema less


    database : 
            create database "use database"

    collection: 
        ->group of documents 
        Syntax to create collection 
                "db.createCollection("collectionName")"


    Insert Documents : 
    ---------------------
        1. insertOne()
                    Syntax : db.collectionName.insertOne({})
        2. insertMany()
                    Syntax : db.collectionName.insertMany([{},{},{},....])
 
    Retrieve Data from collection 
    -----------------------------
        1. findOne() 
        ->Retrieve one document from the collection
                Syntax : db.collectionName.findOne() 
                
                i. filter condition 
                        Syntax: db.collectionName.findOne({filter data}) 
                
                ii. projections 
                        -> to project fields we need to pass value :1 
                        -> if don't project fields we need to pass value :0 

                        Syntax : 
                                db.collectionName.findOne({filter data},{projection qu})
                    

        2.find()
        ->Retrieve all documents from the collection
                Syntax : db.collectionName.find() 
                
                i. filter condition 
                        Syntax: db.collectionName.find({filter data}) 
                
                ii. projections 
                        -> to project fields we need to pass value :1 
                        -> if don't project fields we need to pass value :0 

                        Syntax : 
                                db.collectionName.find({filter data},{projection qu})
                    


    DataTypes  : 
    -----------
            1. string 
            2. number

    Update document : 
    ------------------
        1. updateOne()
                Syntax : 
                        db.collectionName.updateOne({matched field},{update query})
        2. updateMany()



                update operators : 
                -------------------
                    1. $set 
                    db.employeeDetails.updateOne(
                        {name:"Ravi Kumar"},
                                  {
                                      $set:{age:40, companyName:"HCL Tech"}
                                }
                        )



                    2. $unset 
                            ->this operator is used to Delete field

                    db.employeeDetails.updateOne(
                        {name:"Ravi Kumar"},
                                  {
                                      $unset:{age:""}
                                }
                        )

                        
                    3. $inc 

                    db.employeeDetails.updateOne(
                                {},
                                  {
                                      $inc:{salary:20000}
                                }
                        )


                    4. $rename





            Delete documents :
            ----------------- 
                ->this methods are used to Delete document in collection
                1. deleteOne
                        Syntax : db.collectionName.deleteOne({match data})
                        
                2. deleteMany
                        Syntax : db.collectionName.deleteMany({match data})
